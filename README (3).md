## Uploading to GitHub
1. Open the `cldykstra-tax.github.io` repository.
2. Delete or replace the old site files.
3. Upload ALL files and folders from this package.
4. Keep `index.html` at the repository root.
5. Commit the changes.
6. GitHub Pages will redeploy automatically.

The included `CNAME` file already contains:
chrisdykstratax.com
